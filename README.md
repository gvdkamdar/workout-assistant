
# AI-Powered Workout Assistant

An intelligent workout tracking system with a conversational uer interface that uses computer vision and pose estimation to count exercise repetitions in real-time.

## **Features**
- Real-time exercise rep counting using pose estimation
- Voice command interface
- Support for multiple exercises:
  - Bicep Curls
  - Bench Press
  - Lateral Raises
  - Shoulder Press
- Workout logging and tracking
- Audio feedback for rep counting

## **Prerequisites**
- Python 3.8 or above
- Webcam for pose estimation
- Microphone for voice commands

## **Installation**

### **1. Install Required Libraries**
Run the following command to install all dependencies:
```bash
pip install -r requirements.txt
```
## **Usage**

### **Starting the Program**

```bash
python main.py cli
```
### Ending an exercise**

Press Q to end an exercise while on the image window
